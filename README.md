# Résumé du projet
Ce projet réalisé en travail de groupe portant sur l'analyse et l'optimisation d'un poste d'assemblage automobile (P1), en combinant les principes du Lean Manufacturing et les outils de visualisation et d'analyse de données avec Power Bl.
Il a pour objectif principal de diagnostiquer les inefficacités du poste P1 à partir des données réelles de production, 
puis de proposer des actions d’amélioration adaptées.

# Livrables:
- Rapport du projet sous format PDF.
- Fichier EXCEL contenant les tableaux de données à exporter dans Power BI.
- Fichier PBIX contenant les dashboards pour la visualisation et l'exploitation des données.
